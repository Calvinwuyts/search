ECFiddle Redux Instructions

A simple interface for playing with Entity Collection retrieval

Overview

The ECFiddle Redux tool is a stripped-down and simplified version of the ECFiddle prototype. It offers some minimal support for exploring different strategies for querying our documents with entities from the Entity Collection.

It provides some very simple assistance, in the form of:

  1. An autocomplete box, allowing the user to select items from the Entity Collection
  2. An Entity URI text box, displaying the URI of the selected entity.
  3. An Entity Expansion Labels, for displaying the labels stored by the Entity Collection for the selected entity.

The idea is that the user can then cut-and-paste these two strings to combine them into whatever queries he or she desires. Suggested fields to experiment with are supplied in the Candidate Fields panel, though of course the user is free to use whatever fields they deem necessary.

When the user clicks 'Go', a search will be executed using the search string they have created, and the results displayed in the right-hand column. Alternatively, the user can clear the search using the 'Clear' button.
